* Useful Shortcuts

** Files
Open File                  C-x C-f
Open Recent File           C-x b
Close File                 C-x k

** Windows
Switch Between Windows     C-x o
Split Vertically           C-x 3
Split Horizontally         C-x 2
Close Window               C-x 0

** Projects
Open Project File          C-c p f

** Git
Open Magit                 C-c g
